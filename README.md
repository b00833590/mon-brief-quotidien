# mon-brief-quotidien

[![Brief quotidien automatique](https://github.com/b00833590/mon-brief-quotidien/actions/workflows/brief-quotidien.yml/badge.svg)](https://github.com/b00833590/mon-brief-quotidien/actions/workflows/brief-quotidien.yml)

Brief quotidien automatisé — résumé de mes newsletters financières (Aktionnaire, Arcos, Les Echos, FT), généré et mis à jour chaque jour via Python et l'API Gemini. Tourne automatiquement dans le cloud (GitHub Actions) à 8h et 13h heure de Paris — voir `RUNBOOK.md` pour l'exploitation.
